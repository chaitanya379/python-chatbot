# 🤖 Python Chatbot

A simple Python Chatbot developed as part of the CodeAlpha Python Programming Internship. This chatbot interacts with users through text-based conversations and provides predefined responses to common queries.

## 📌 Features

* Interactive text-based conversation
* Responds to common greetings and questions
* User-friendly interface
* Fast and efficient responses
* Easy to customize with new commands and responses

## 🛠️ Technologies Used

* Python 3

## 🚀 How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Run the Python file:

```bash
python task4.py
```

4. Start chatting with the bot.

## 💬 Example Conversation

User: Hello

Bot: Hi! How can I help you?

User: How are you?

Bot: I'm doing great! Thanks for asking.

User: Bye

Bot: Goodbye! Have a nice day.
